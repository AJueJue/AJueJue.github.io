11. 盛最多水的容器

难度中等4052

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

代码（c++）

该题我用的是双指针+贪心做的

我们知道长方形的面积=长*宽 即(r-l)*min(height[r],height[l])

用双指针比较两个边长的大小来取得合理的长宽

时间复杂度O(n) n为数组长度

空间复杂度O(1)
